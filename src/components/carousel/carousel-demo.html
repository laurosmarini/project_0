How to Use These Carousels

  • Mouse Users: Click on dot indicators or arrow buttons to navigate
  • Keyboard Users: Tab to focus on indicators, then press Enter or Space to activate
  • Touch Devices: Tap indicators or swipe on supporting devices
  • Screen Readers: Navigate by landmarks and use the tab navigation pattern
  • Auto-play Carousel: Hover or focus any element to pause automatic advancement

Accessibility Features

  • WCAG 2.1 AA compliant color contrast ratios
  • Complete keyboard navigation support
  • Proper ARIA roles and labels for screen readers
  • Respects reduced motion preferences
  • Focus management and visible focus indicators
